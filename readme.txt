Add additional fee or discount to your cart/checkout.

== Description ==

This plugin lets you add additional fee to your checkout page. Add any addtional charges like - handling charge with ease. Install the plugin and enter your extra charge amount and name of the fee.

= Basic Features =

* Apply additional fee to your cart. ( Like handling charges or Insurance charges)
* Apply special discounts to your cart.
* Apply fee/discount if cart total crosses an amount ( cart minimum )
* Apply fee/discount if cart total remains below an amount ( cart maximum )

With the above features you can handle different cart scenarios and apply fee(s) according to that. 

= Most common scenario(s) we have seen =

* Apply a fee while minimum purchase amount was not fulfilled by a customer.
* Apply a discount if customer purchases for certain amount or more.

== Installation ==

Manual

1. Upload the plugin folder to the '/wp-content/plugins/' directory .

2. Activate the plugin through the 'Plugins' menu in WordPress.

Automatic

1. Go to plugin list from 'dashboard'. Search our plugin and click to install

2. Activate it.
